Arrays in C#

The Basics

An array is a collection of elements of the same type, stored in contiguous memory locations. Here's an example of how to declare and initialize an array in C#:


int[] numbers = { 1, 2, 3, 4, 5 };

In this example, we are declaring an array of integers named numbers, and initializing it with 5 values. The array index starts at 0, so the first element is numbers[0], the second element is numbers[1], and so on.

You can also declare an array without initializing it, and then assign values to it later, like this:


int[] numbers = new int[5];

numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

In this example, we are declaring an array of integers named numbers, with a length of 5. Then, we are assigning values to each element of the array.

Types of Arrays

In C#, there are several types of arrays:

  • Single-Dimensional Arrays: These are the most common type of arrays, where the elements are arranged in a single row or column.
  • Multidimensional Arrays: These are arrays with more than one dimension, where the elements are arranged in rows and columns.
  • Jagged Arrays: These are arrays of arrays, where each element of the array is itself an array.

Single-Dimensional Arrays

Here's an example of a single-dimensional array:


int[] numbers = { 1, 2, 3, 4, 5 };

You can access the elements of a single-dimensional array using an index, like this:


int thirdNumber = numbers[2]; // returns 3

Multidimensional Arrays

Here's an example of a two-dimensional array:


int[,] matrix = new int[3, 3];

matrix[0, 0] = 1;
matrix[0, 1] = 2;
matrix[0, 2] = 3;

matrix[1, 0] = 4;
matrix[1, 1] = 5;
matrix[1, 2] = 6;

matrix[2, 0] = 7;
matrix[2, 1] = 8;
matrix[2, 2] = 9;

In this example, we are declaring a two-dimensional array of integersnamed matrix, with a length of 3 rows and 3 columns. We are then assigning values to each element of the array using two indices.

You can access the elements of a multidimensional array using multiple indices, like this:


int fifthElement = matrix[1, 1]; // returns 5

Jagged Arrays

Here's an example of a jagged array:


int[][] jaggedArray = new int[3][];

jaggedArray[0] = new int[] { 1, 2 };
jaggedArray[1] = new int[] { 3, 4, 5 };
jaggedArray[2] = new int[] { 6, 7, 8, 9 };

In this example, we are declaring an array of arrays named jaggedArray. Each element of the array is itself an array of integers. The first element has 2 elements, the second element has 3 elements, and the third element has 4 elements.

You can access the elements of a jagged array using multiple indices, like this:


int secondElementThirdValue = jaggedArray[1][2]; // returns 5

These are just a few examples of how to use arrays in C#. There are many more ways to create, manipulate, and use arrays in your programs.
