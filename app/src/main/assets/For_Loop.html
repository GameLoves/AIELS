For Loops in C#

The for loop is used to execute a block of code repeatedly, for a specified number of times. Here's the basic syntax of a for loop:


for (initialization; condition; increment/decrement)
{
    // code to execute while condition is true
}

The for loop consists of three parts: the initialization statement, the condition statement, and the increment/decrement statement. The initialization statement is executed once, before the loop starts. The condition statement is evaluated at the beginning of each iteration, and if it is true, the code inside the loop is executed. The increment/decrement statement is executed at the end of each iteration. Here's an example:


for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}

In this example, the loop executes 5 times, with i starting at 0 and incrementing by 1 each time. The output of this code is:


0
1
2
3
4

There are several variations of the for loop in C#, including the foreach loop, which is used to iterate over collections, and the for loop with multiple initialization statements. Here's an example of a foreach loop:


int[] numbers = { 1, 2, 3, 4, 5 };

foreach (int number in numbers)
{
    Console.WriteLine(number);
}

In this example, the loop iterates over the array of integers, outputting each value to the console. The output of this code is:


1
2
3
4
5

Here's an example of a for loop with multiple initialization statements:


for (int i = 0, j = 10; i < j; i++, j--)
{
    Console.WriteLine($"i = {i}, j = {j}");
}

In this example, the loop executes 5 times, with two initialization statements: i starts at 0 and j starts at 10. Each time the loop iterates, i increments by 1 and j decrements by 1. The output of this code is:


i = 0, j = 10
i = 1, j = 9
i = 2, j = 8
i = 3, j = 7
i = 4, j = 6

These are just a few examples of what you can do with for loops in C#. There aremany more ways to use them, and you can also combine them with other programming constructs to create more complex logic.
