Strings in C#

C# provides a string keyword to represent a sequence of characters. Here are some common operations performed on strings:

  • Length - Returns the number of characters in the string.
  • IndexOf - Returns the position of the first occurrence of a specified character or substring in the string.
  • Substring - Returns a substring of the string, starting from a specified position and with a specified length.
  • Replace - Returns a new string in which all occurrences of a specified string or character in the current string are replaced with another specified string or character.
  • ToLower - Returns a copy of the string converted to lowercase.
  • ToUpper - Returns a copy of the string converted to uppercase.
  • Trim - Returns a copy of the string with all leading and trailing white-space characters removed.
  • Split - Returns an array of strings that contains the substrings in the original string that are delimited by specified characters.
  • Concat - Concatenates two or more strings.

Examples

Here are some examples of using strings in C#:


string s1 = "Hello, world!";
string s2 = "C# is awesome!";

// Length
int length1 = s1.Length; // length1 = 13
int length2 = s2.Length; // length2 = 15

// IndexOf
int index1 = s1.IndexOf("world"); // index1 = 7
int index2 = s2.IndexOf("Java"); // index2 = -1

// Substring
string sub1 = s1.Substring(7, 5); // sub1 = "world"
string sub2 = s2.Substring(0, 2); // sub2 = "C#"

// Replace
string replace1 = s1.Replace("world", "everyone"); // replace1 = "Hello, everyone!"
string replace2 = s2.Replace("awesome", "amazing"); // replace2 = "C# is amazing!"

// ToLower and ToUpper
string lower1 = s1.ToLower(); // lower1 = "hello, world!"
string upper2 = s2.ToUpper(); // upper2 = "C# IS AWESOME!"

// Trim
string trim1 = "   hello   ".Trim(); // trim1 = "hello"

// Split
string s3 = "The quick brown fox";
string[] words = s3.Split(' '); // words = {"The", "quick", "brown", "fox"}

// Concat
string concat1 = string.Concat(s1, " ", s2); // concat1 = "Hello, world! C# is awesome!"

String Interpolation

C# also provides a feature called string interpolation, which allows you to embed expressions inside string literals. To use string interpolation, you prefix the string literal with the $ symbol and then use curly braces {} to enclose the expression you want to embed. Here's an example:


string name = "John";
int age = 30;
string message = $"My name is {name} and I am {age} years old.";
// message = "My name is John and I am 30 years old."

Accessing Strings

You can access individual characters in a string using the square bracket notation. The index of the first character in a string is 0, and the index of the last character is the length of the string minus 1. Here's an example:


string s = "Hello, world!";
char first = s[0]; // first = 'H'
char last = s[s.Length - 1]; // last = '!'

Note that attempting to access an index outside the bounds of the string will result in an IndexOutOfRangeException.

Special Characters

C# supports various special characters that can be included in string literals. Here are some examples:

  • \n - Newline
  • \r - Carriage return
  • \t - Tab
  • \" - Double quote
  • \' - Single quote
    • \\ - Backslash

Here's an example of using special characters:


string s = "This is a \"quoted\" string.\nIt has a newline and a\ttab.";
// s = "This is a "quoted" string.
// It has a newline and a    tab."

These are just a few examples of what you can do with strings in C#. There are many more methods available, and you can also combine them to create more complex operations.
