C# Data Types

Basic Data Types

C# is a strongly-typed language, which means that every variable and constant must be declared with a specific data type. The following are the basic data types available in C#:

  • bool: A boolean value that can be either true or false.
  • byte: An 8-bit unsigned integer that can hold values from 0 to 255.
  • char: A single Unicode character that can be any character in the Unicode standard, including letters, digits, and symbols.
  • decimal: A decimal value with 28-29 significant digits that is used for financial and monetary calculations.
  • double: A double-precision floating-point value that can hold values with up to 15-16 digits.
  • float: A single-precision floating-point value that can hold values with up to 7 digits.
  • int: A 32-bit signed integer that can hold values from -2,147,483,648 to 2,147,483,647.
  • long: A 64-bit signed integer that can hold values from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.
  • sbyte: An 8-bit signed integer that can hold values from -128 to 127.
  • short: A 16-bit signed integer that can hold values from -32,768 to 32,767.
  • uint: A 32-bit unsigned integer that can hold values from 0 to 4,294,967,295.
  • ulong: A 64-bit unsigned integer that can hold valuesfrom 0 to 18,446,744,073,709,551,615.
  • ushort: A 16-bit unsigned integer that can hold values from 0 to 65,535.

In addition to these basic data types, C# also supports more complex data types such as arrays, structures, and classes. These data types allow you to organize and manipulate data in more sophisticated ways.

Declaring Variables

To declare a variable in C#, you need to specify its data type and name:

int age;

This declares a variable named age of type int.

You can also assign an initial value to the variable when you declare it:

int age = 30;

This declares a variable named age of type int and initializes it with a value of 30.

Type Conversion

Sometimes you may need to convert a variable from one data type to another. This is known as type conversion. C# provides two ways to perform type conversion: implicit and explicit.

  • Implicit conversion: This occurs when the compiler automatically converts a value from one data type to another. For example, you can assign an int value to a double variable without any explicit conversion.
  • Explicit conversion: This occurs when you manually convert a value from one data type to another using a type cast. For example, you can convert a double value to an int using the following code:
double d = 3.14;
int i = (int)d;

Here, the double value 3.14 is explicitly converted to an int using a type cast.

It's important to note that type conversion can sometimes result in a loss of precision or data. For example, converting a double to an int will result in the loss of any fractional part of the value.
