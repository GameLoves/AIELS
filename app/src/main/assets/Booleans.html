Booleans in C#

C# provides a bool keyword to represent a Boolean value, which can be either true or false. Booleans are commonly used in conditional statements to control the flow of program execution. Here are some common operations performed on Booleans:

  • && - Logical AND operator. Returns true if both operands are true.
  • || - Logical OR operator. Returns true if at least one of the operands is true.
  • ! - Logical NOT operator. Returns true if the operand is false, and false if the operand is true.

Examples

Here are some examples of using Booleans in C#:


bool a = true;
bool b = false;

// Logical AND
bool result1 = a && b; // result1 = false

// Logical OR
bool result2 = a || b; // result2 = true

// Logical NOT
bool result3 = !a; // result3 = false
bool result4 = !b; // result4 = true

These are just a few examples of what you can do with Booleans in C#. There are many more logical operators available, and you can also combine them to create more complex operations.
