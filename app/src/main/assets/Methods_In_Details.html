Methods in C#

The Basics

A method is a block of code that performs a specific task. In C#, you define a method inside a class. Here's an example of a simple method:


public void SayHello()
{
    Console.WriteLine("Hello!");
}

In this example, we are defining a method named SayHello that does not return a value (the return type is void). The method simply writes "Hello!" to the console.

Types of Methods

In C#, there are several types of methods:

  • Static Methods: These are methods that belong to a class, rather than an instance of the class.
  • Instance Methods: These are methods that belong to an instance of a class.
  • Constructors: These are special methods that are called when an object of a class is created.
  • Destructors: These are special methods that are called when an object of a class is destroyed.

Static Methods

Here's an example of a static method:


public static int Add(int a, int b)
{
    return a + b;
}

In this example, we are defining a static method named Add that takes two integer parameters and returns their sum.

You can call a static method using the class name, like this:


int sum = MyClass.Add(3, 4); // returns 7

Instance Methods

Here's an example of an instance method:


public void SayHello()
{
    Console.WriteLine("Hello!");
}

In this example, we are defining an instance method named SayHello that does not return a value. You can call an instance method on an object of the class, like this:


MyClass myObject = new MyClass();
myObject.SayHello(); // writes "Hello!" to the console

Method Parameters

In C#, you can pass parameters to a method. This allows you to customize the behavior of the method based on the input values. Here's an example:


public void SayHello(string name)
{
    Console.WriteLine("Hello, " + name + "!");
}

In this example, we are defining an instance method named SayHello that takes a string parameter named name. The method writes "Hello, " followed by the value of name to the console.

You can call this method and pass a value for the name parameter, like this:


MyClass myObject = new MyClass();
myObject.SayHello("John"); // writes "Hello, John!" to the console

Method Overriding

Method overriding is a feature of object-oriented programming that allows a subclass to provide a specific implementation of a method that is already defined in its superclass. Here's an example:


public class Animal
{
    public virtual void MakeSound()
    {
        Console.WriteLine("The animal makes a sound.");
    }
}

public class Cat : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("The cat meows.");
    }
}

In this example, we define a base class named Animal with a virtual method named MakeSound. We then define a subclass named Cat which overrides the MakeSound method with its own implementation.

You can create an instance of the Cat class and call the MakeSound method to see the overridden behavior:


Cat myCat = new Cat();
myCat.MakeSound(); // writes "The cat meows." to the console

Method Overloading

Method overloading is a feature of C# that allows you to define multiple methods with the same name but different parameters. Here's an example:


public class Calculator
{
    public int Add(int a, int b)
    {
        return a + b;
    }

    public double Add(double a, double b)
    {
        return a + b;
    }
}

In this example, we define a class named Calculator with two methods named Add. The first method takes two integer parameters and returns their sum, while the second method takes two double parameters and returns their sum.

You can call either of these methods, and the correct one will be selected based on the parameter types:


Calculator myCalc = new Calculator();
int sum1 = myCalc.Add(3, 4); // returns 7
double sum2 = myCalc.Add(2.5, 3.5); // returns 6.0
