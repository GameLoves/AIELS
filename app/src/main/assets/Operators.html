Operators in C#

C# provides several types of operators to perform various types of operations on operands. Here are the different types of operators:

  • Arithmetic operators
  • Relational operators
  • Logical operators
  • Assignment operators
  • Conditional operators
  • Bitwise operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on operands. Here are the arithmetic operators in C#:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ++ (Increment)
  • -- (Decrement)

Here are some examples:


int x = 10;
int y = 5;

int result = x + y; // result = 15
result = x - y; // result = 5
result = x * y; // result = 50
result = x / y; // result = 2
result = x % y; // result = 0
x++; // x = 11
y--; // y = 4

Relational Operators

Relational operators are used to compare two operands. Here are the relational operators in C#:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)

Here are some examples:


int x = 10;
int y = 5;

bool result = x == y; // result = false
result = x != y; // result = true
result = x > y; // result = true
result = x < y; // result = false
result = x >= y; // result = true
result = x <= y; // result = false

Logical Operators

Logical operators are used to perform logical operations on boolean values. Here are the logical operators in C#:

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)

    • Here are some examples:

    
bool a = true;
bool b = false;

bool result = a && b; // result = false
result = a || b; // result = true
result = !a; // result = false

    Assignment Operators

    Assignment operators are used to assign a value to a variable. Here are the assignment operators in C#:

    • = (Assignment)
    • += (Addition assignment)
    • -= (Subtraction assignment)
    • *= (Multiplication assignment)
    • /= (Division assignment)
    • %= (Modulus assignment)

    Here are some examples:

    
int x = 10;

x += 5; // x = 15
x -= 3; // x = 12
x *= 2; // x = 24
x /= 4; // x = 6
x %= 2; // x = 0

    Conditional Operators

    Conditional operators are used to perform conditional operations. Here is the conditional operator in C#:

    • ?: (Conditional operator)

    Here is an example:

    
int x = 10;
int y = 5;

int max = x > y ? x : y; // max = 10

    Bitwise Operators

    Bitwise operators are used to perform bitwise operations on operands. Here are the bitwise operators in C#:

    • & (Bitwise AND)
    • | (Bitwise OR)
    • ^ (Bitwise XOR)
    • ~ (Bitwise NOT)
    • << (Left shift)
    • >> (Right shift)

    Bitwise operators are beyond the scope of this basic introduction to C# operators, but they are useful for performing operations at the bit level and for working with binary data.
