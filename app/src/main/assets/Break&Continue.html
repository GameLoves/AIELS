Break and Continue Statements in C#

The break and continue statements are used to control the flow of execution in loops. Here's an explanation of how they work:

  • The break statement is used to exit a loop prematurely. When the break statement is encountered inside a loop, the loop is immediately terminated, and the program continues executing the code that comes after the loop. Here's an example:

for (int i = 0; i < 10; i++)
{
    if (i == 5)
    {
        break;
    }

    Console.WriteLine(i);
}

In this example, the for loop executes 10 times, with i starting at 0 and incrementing by 1 each time. When i reaches 5, the break statement is encountered, and the loop is terminated prematurely. The output of this code is:


0
1
2
3
4
  • The continue statement is used to skip the current iteration of a loop and continue with the next iteration. When the continue statement is encountered inside a loop, the program skips the rest of the code inside the loop for the current iteration, and continues with the next iteration. Here's an example:

for (int i = 0; i < 10; i++)
{
    if (i == 5)
    {
        continue;
    }

    Console.WriteLine(i);
}

In this example, the for loop executes 10 times, with i starting at 0 and incrementing by 1 each time. When i reaches 5, the continue statement is encountered, and the program skips the rest of the code inside the loop for that iteration, and continues with the next iteration. The output of this code is:


0
1
2
3
4
6
7
8
9

These are just a few examples of what you can do with the break and continue statements in C#. There are many more ways to use them, and you can also combine them with other programming constructs to create more complex logic.
