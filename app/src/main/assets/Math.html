Math in C#

C# provides a Math class that contains various static methods for performing mathematical operations. Here are some of the methods provided by the Math class:

  • Abs - Returns the absolute value of a number.
  • Acos - Returns the arccosine of a number.
  • Asin - Returns the arcsine of a number.
  • Atan - Returns the arctangent of a number.
  • Atan2 - Returns the angle whose tangent is the quotient of two specified numbers.
  • Ceiling - Returns the smallest integer greater than or equal to a number.
  • Cos - Returns the cosine of a number.
  • Cosh - Returns the hyperbolic cosine of a number.
  • Exp - Returns e raised to the specified power.
  • Floor - Returns the largest integer less than or equal to a number.
  • IEEERemainder - Returns the remainder resulting from the division of two specified numbers.
  • Log - Returns the natural (base e) logarithm of a specified number.
  • Log10 - Returns the base 10 logarithm of a specified number.
  • Max - Returns the larger of two specified numbers.
  • Min - Returns the smaller of two specified numbers.
  • Pow - Returns a specified number raised to the specified power.
  • Round - Rounds a number to the nearest integer or to a specified number of decimal places.
  • Sign - Returns a value indicating the sign of a number.
  • Sin - Returns the sine of a number.
  • Sinh - Returns the hyperbolic sine of a number.
  • Sqrt - Returns the square root of a specified number.
  • Tan - Returns the tangent of a number.
  • Tanh - Returns the hyperbolic tangent of a number.
  • Truncate - Calculates the integral part of a specified number.

Examples

Here are some examples of using the Math class:


double x = 3.14;
double y = -2.5;

// Absolute value
double absX = Math.Abs(x); // absX = 3.14
double absY = Math.Abs(y); // absY =2.5

// Ceiling and floor
double ceilX = Math.Ceiling(x); // ceilX = 4
double floorY = Math.Floor(y); // floorY = -3

// Maximum and minimum
double maxXY = Math.Max(x, y); // maxXY = 3.14
double minXY = Math.Min(x, y); // minXY = -2.5

// Rounding
double roundX = Math.Round(x); // roundX = 3
double roundY = Math.Round(y, 1); // roundY = -2.5

// Truncating
double truncX = Math.Truncate(x); // truncX = 3
double truncY = Math.Truncate(y); // truncY = -2

These are just a few examples of what you can do with the Math class in C#. There are many more methods available, and you can also combine them to create more complex calculations.
