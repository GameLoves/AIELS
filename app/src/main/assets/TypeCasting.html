Type Casting in C#

C# provides the ability to convert a value of one data type to another data type. This is known as type casting. There are two types of type casting in C#:

  • Implicit Type Casting
  • Explicit Type Casting

Implicit Type Casting

Implicit type casting is done automatically by the compiler when a conversion from a smaller data type to a larger data type is required. For example:

int num = 10;
double d = num; // Implicit type casting from int to double

In this example, the value of the int variable num is implicitly cast to a double variable d.

Explicit Type Casting

Explicit type casting is done by the programmer when a conversion from a larger data type to a smaller data type is required. For example:

double d = 10.5;
int num = (int)d; // Explicit type casting from double to int

In this example, the value of the double variable d is explicitly cast to an int variable num.

It is important to note that when casting from a larger data type to a smaller data type, there is a risk of data loss. For example, when casting from a double to an int, any decimal places will be lost.

Conclusion

Type casting is an important concept in C# that allows for the conversion of values from one data type to another. Implicit type casting is done automatically by the compiler, while explicit type casting is done by the programmer. When casting from a larger data type to a smaller data type, there is a risk of data loss.
