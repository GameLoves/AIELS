While Loops in C#

The while loop is used to execute a block of code repeatedly, as long as a specified condition is true. Here's the basic syntax of a while loop:


while (condition)
{
    // code to execute while condition is true
}

The code inside the curly braces is executed repeatedly as long as the condition inside the parentheses is true. Here's an example:


int i = 0;

while (i < 5)
{
    Console.WriteLine(i);
    i++;
}

In this example, the value of i starts at 0, and the code inside the while loop is executed repeatedly as long as i is less than 5. Each time the code is executed, the value of i is incremented by 1, and the current value of i is output to the console. The output of this code is:


0
1
2
3
4

You can also use the while loop with other programming constructs, such as the break and continue statements. Here's an example that uses both:


int num = 0;

while (num < 10)
{
    num++;

    if (num == 5)
    {
        continue;
    }

    if (num == 8)
    {
        break;
    }

    Console.WriteLine(num);
}

In this example, the loop continues to execute as long as num is less than 10. Inside the loop, num is incremented by 1 each time, and then two conditional statements are evaluated. The first conditional uses the continue statement to skip the current iteration of the loop if num is 5. The second conditional uses the break statement to exit the loop if num is 8. The output of this code is:


1
2
3
4
6
7

These are just a few examples of what you can do with while loops in C#. There are many more ways to use them, and you can also combine them with other programming constructs to create more complex logic.
