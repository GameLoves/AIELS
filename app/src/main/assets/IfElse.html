If...Else Statements in C#

The if...else statement is used to execute a block of code if a specified condition is true, and another block of code if the condition is false. Here are the different types of if...else statements in C#:

Single If...Else Statement

The simplest form of if...else statement is a single if statement followed by an else statement. Here's an example:


int x = 10;

if (x > 5)
{
    Console.WriteLine("x is greater than 5.");
}
else
{
    Console.WriteLine("x is less than or equal to 5.");
}

In this example, the condition in the if statement is true, so the first block of code is executed, which outputs "x is greater than 5."

If...Else If...Else Statement

You can also chain multiple if statements together using the else if keyword. Here's an example:


int age = 20;

if (age < 18)
{
    Console.WriteLine("You are a minor.");
}
else if (age >= 18 && age < 65)
{
    Console.WriteLine("You are an adult.");
}
else
{
    Console.WriteLine("You are a senior citizen.");
}

In this example, the age is 20, so the second condition is true, and the corresponding block of code is executed, which outputs "You are an adult."

Nested If...Else Statements

You can also nest if...else statements inside each other to check for more complex conditions. Here's an example:


int num = 10;

if (num > 0)
{
    if (num % 2 == 0)
    {
        Console.WriteLine("The number is positive and even.");
    }
    else
    {
        Console.WriteLine("The number is positive and odd.");
    }
}
else if (num < 0)
{
    Console.WriteLine("The number is negative.");
}
else
{
    Console.WriteLine("The number is zero.");
}

In this example, the number is positive and even, so the corresponding block of code is executed, which outputs "The number is positive and even."

These are just a few examples of what you can do with if...else statements in C#. There are many more ways to use them, and you can also combine them with other programming constructs to create more complex logic.
